3DS SHELL EXTENSION
===================

  ABOUT

This is an extension for Windows Explorer that displays pictures of game
cartridges for .3ds files, which are Ninendo 3DS cartridge dumps.

  INSTALLATION
  
Run install.bat to install (double-click it). That should be it.
Make sure you install a version that fits your system: x64 for 64-bit
systems, and x32 for 32-bit systems. You can also uninstall the extension
with uninstall.bat.
